Бриф = паспорт + журнал + задание на орган. В новом чате: вставь из буфера и приложи актуальный index.html.
Полка
вскрытые шары · сад Латимера жив с 1972-го
«На стол» возвращает шар к жизни: читать ленту, выгружать, даже продлить срок. Текущая сессия при этом сама встаёт на полку. Хранится до 6 шаров, старейший уступает место.

Свои субличности хранятся на этом устройстве и доступны обоим слотам. Пресеты неизменяемы, но их можно взять за основу.
Настройки
ключ api · хранится только на этом устройстве
Anthropic API key
Внутри Claude (артефакт) ключ не нужен — оставь пустым. Для GitHub Pages вставь ключ с console.anthropic.com: он ляжет в localStorage и уходит только на api.anthropic.com. Не публикуй файл с вписанным ключом.
